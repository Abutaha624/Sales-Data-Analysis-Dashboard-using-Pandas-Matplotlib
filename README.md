# Sales-Data-Analysis-Dashboard-using-Pandas-Matplotlib
# 📊 مشروع تحليل بيانات المبيعات

يقوم هذا المشروع بتحليل بيانات المبيعات باستخدام بايثون (Pandas و NumPy و Matplotlib).

## 🔍 نظرة عامة على المشروع
ينفذ السكربت عمليات تنظيف البيانات، والتحليل الإحصائي، وتقسيم العملاء، وإنشاء التصورات البيانية لبيانات المبيعات.

## 🧹 معالجة البيانات
- إزالة القيم المفقودة  
- تحويل تواريخ الطلبات إلى صيغة تاريخ (Datetime)  
- تجهيز البيانات للتحليل  

## 📈 التحليلات المنفذة
1. إجمالي المبيعات حسب الفئة  
2. إجمالي ومتوسط الأرباح حسب المنطقة  
3. تقسيم العملاء بناءً على إجمالي مشترياتهم (Quartiles)  
4. قياس العلاقة بين الكمية والربح  
5. تحليل أداء القطاعات (إحصائيات المبيعات والأرباح)  
6. تحليل اتجاه الأرباح الشهرية عبر الزمن  

## 📊 الرسوم البيانية المُنشأة
- مخطط أعمدة: المبيعات حسب الفئة  
- مخطط دائري: الأرباح حسب المنطقة  
- مخطط مبعثر: العلاقة بين الكمية والربح  
- مخطط خطي: اتجاه الأرباح الشهرية  
- مخطط أعمدة: أداء المبيعات حسب القطاع  

## 🛠️ التقنيات المستخدمة
- Python  
- Pandas  
- NumPy  
- Matplotlib  

## 📁 المخرجات
يقوم السكربت بحفظ جميع الرسوم البيانية تلقائيًا بصيغة PNG.
